Habilidades

Com experiência na criação de interfaces dinâmicas e responsivas, sou apaixonada por transformar ideias criativas em experiências digitais envolventes. Minha jornada no desenvolvimento web me permitiu dominar tecnologias essenciais como JavaScript, HTML, CSS, TypeScript e Java, resultando em projetos robustos e de alta performance. JavaScript: Domínio de frameworks e bibliotecas modernas, incluindo React, para construção de interfaces interativas e eficientes. Utilização de TypeScript para maior segurança e escalabilidade no desenvolvimento de aplicações. HTML & CSS: Habilidade em criar layouts responsivos e acessíveis, utilizando as melhores práticas de design responsivo. Java: Experiência em integrar front-end com back-end robusto, utilizando Spring Framework e outras tecnologias Java. UX/UI Design: Foco em criar experiências de usuário intuitivas e esteticamente agradáveis, colaborando estreitamente com equipes de design. Estou sempre em busca de novas oportunidades para expandir meus conhecimentos e contribuir para projetos inovadores. Meu objetivo é continuar evoluindo como desenvolvedor front-end, aproveitando as últimas tendências e tecnologias para criar soluções impactantes.
